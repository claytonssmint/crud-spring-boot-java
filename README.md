# API de Produtos

POST	/products	Cadastra um produto
GET	/products	Retorna uma lista de produtos
GET	/products/{id}	Retorna o produto por id
DELETE	/products/{id}	Deleta produto por id
UPDATE	/products/{id}	Atualiza o produto por id
